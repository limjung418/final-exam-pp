웹프로그래밍 학습 내용

HTML, CSS, JavaScript를 공부한 기록

소개 학습 내용 팀 프로젝트 블로그 링크

HTML

HTML은 웹페이지의 구조를 만드는 언어입니다. 제목, 문단, 이미지, 링크, 표, 입력 폼 등을 사용하여 웹페이지의 기본 틀을 만들 수 있습니다.

CSS

CSS는 웹페이지의 디자인을 담당합니다. 글자 색, 배경색, 여백, 정렬, 박스 디자인 등을 적용하여 보기 좋은 웹페이지를 만들 수 있습니다.

JavaScript

JavaScript는 웹페이지에 동적인 기능을 추가하는 언어입니다. 변수, 연산자, 조건문, 함수, 이벤트, DOM 조작을 학습했습니다.
